## How It Works

1. Choose **Rock**, **Paper**, or **Scissors**.
2. The computer randomly selects one of the three.
3. The winner is decided using standard rules:
   - Rock beats Scissors
   - Scissors beat Paper
   - Paper beats Rock
4. The score updates automatically after every round.
